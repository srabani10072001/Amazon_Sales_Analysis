# 📊 Amazon Sales Dashboard

## 🔹 Overview

This project analyzes Amazon sales data using Power BI.

## 🔹 Key Metrics

* Total Sales: 1.26M
* Total Orders: 2K
* Avg Order Value: 630+
* Total Quantity: 2K

## 🔹 Dashboard Features

* Sales trend analysis (Line Chart)
* State-wise performance (Bar Chart)
* Category-wise distribution (Pie Chart)
* Order status breakdown (Donut Chart)
* Interactive slicer for filtering

## 🔹 Tools Used

* Power BI
* SQL
* Excel
* Python (Pandas, NumPy)
